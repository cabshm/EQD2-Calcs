# Re-irradiation EQD2 Calculator (HTML App)

Standalone static HTML/CSS/JavaScript calculator for re-irradiation BED and EQD2.

## Files
- `index.html`
- `style.css`
- `app.js`

## Publish on GitHub Pages
1. Create a new GitHub repo for this calculator.
2. Upload the files in this folder.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** branch and **/(root)**.
6. Save.

Your app will be published at:
`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`\n
