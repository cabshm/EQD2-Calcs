# Re-irradiation EQD2 Calculator (HTML App) — v1.1

Adds a time-correction / recovery factor applied to Course 1 before EQD2 summation.

## New outputs
- Course 1 corrected EQD2
- Total EQD2 (time-corrected)

## Interpretation
If recovery factor = 25%, then Course 1 contributes 75% of its original EQD2 to the final sum.

## Files
- index.html
- style.css
- app.js

## Publish on GitHub Pages
1. Create a new GitHub repo for this calculator.
2. Upload the files in this folder.
3. In GitHub, go to Settings → Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select main branch and /(root).
6. Save.
